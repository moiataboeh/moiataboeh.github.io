賽夏族傳統服飾之織布整經計算工具

本工具用於支援賽夏族傳統「短衣」與「長衣」之整經計算，方便進行經線預估與織作準備。

基礎公式說明：
1.原始經線條數 (W) = ((a + g) / 2) * 12
→ 依肩寬與鋼扣密度所計算出的初始條數。肩寬加預留空間除以 2，再乘以鋼扣測量得知每公分 12 條經線。
2.原始經線長度 (L) = 2[(2r + 2g) + (2S + 2g)]
→ 依衣長與織片數所計算出的初始長度。短衣與長衣各兩片，並加上預留空間。
3.經線條數 (W₁) = ⌈W / x⌉ * x
→ 因前面已預留空間(g)，此處經線條數只需向上取整為捆線數（x）的倍數，以方便整經時計算。
4.經線長度 (L₁) = ⌈L / y⌉ * y
→ 經線長度向上取整為整經柱距（y）的倍數，並適度多估算，避免無法完成織作。
5.整經拉線次數 (C₁) = W₁ / x
→ 計算整經時拉捆線的總次數，以利整經計算。
6.整經繞柱次數 (C₂) = L₁ / y
→ 計算經線在整經架上往返的總繞線次數，以利整經計算。
7.穿綜絲數 (H) = W₁ / 8
→ 經線除以8後的結果，為一組8條的分組數量，完整為整數，小數點則由自行評估是否整數化，後續檢視機台的提綜片上的綜絲是否足夠。

※ 備註 ※
1.多使用15至16dpi鋼扣，且實測估算每公分約12條經線。
2.綜絲計算以 8 條為一組，順序為 1-2-3-4-5-6-3-4，是賽夏族傳統織布的特徵排法。
3. 各數值可依個人尺寸與衣型進行調整，計算結果為整經參考基準。
